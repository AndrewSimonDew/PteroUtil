# Pterodactyl Utility
For simplicity

A Minecraft Velocity plugin that allows managing the Pterodactyl gameserver panel through its REST API.

- Usage /serverctl \<start|stop|restart|status\> \<servername\>

- Permission: pteroutil.serverctl
